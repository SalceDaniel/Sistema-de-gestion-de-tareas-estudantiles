El programa hace uso de 2 archivos:
1. listaEstudiantes.dat que guarda: la información de docentes
2. listaEstudiantes.dat que guarda los datos de cada Estudiante, incluyendo:
        - Tareas y subtareas de cada estudiante
        - Usuarios, Contraseñas
        - Asignaturas

Las notificaciones se realizan mediante el correo electrónico,
 y se da la opción de notificar cada vez que se ingresa a la aplicación,
  en caso de tener las notificaciones activadas

Para ejecutarel programa son necesarias las siguientes librerías:
    Gráficos para reportes:
        pip install PyQt6 PyQt6-Charts
